# Using seismicity to constrain dike opening inversion models from fiber optic geodesy
 This is a MS thesis in Geophysics from the University of Iceland, created by Robin Schindfessel. Abstract : We will use Low Frequency Distributed Acoustic Sensing data and earthquake locations from the recent events in the Reykjanes peninsula and develop inversion schemes that derive stress change from earthquake locations to constrain dike opening. Since previous results show a difference with earthquake location, the idea is to use earthquake catalogues and build an inversion model constrained by it. The Dietrich 1994 seismicity rate theory will be used to map observed seismicity to stress changes adjacent to the dikes. Thus, the research question will be: “Is it possible to constrain dike opening model from LFDAS inversion to be consistent with earthquakes location ?”. Modelling will be carried out using matlab. We assume a deflating Mogi-point source for the sill and dislocation with opening of the dike near the Sundhnúkur crater rows. The main goal of this project is to give tools for joint modelling of seismic and LFDAS data and monitoring of future events in the area.
